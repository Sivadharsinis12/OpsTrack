# OpsTrack – Internal Issue Escalation & Resolution Tracker

OpsTrack is a lightweight internal issue tracking system built using pure HTML, CSS, and Vanilla JavaScript.

It allows teams to:
- Report operational issues
- Track issue status (Pending, In Progress, Resolved)
- View real-time dashboard statistics
- Monitor category-based trends
- View issue history
- Paginate large datasets

All data is stored using browser localStorage (no backend required).

---

## 🚀 Features

- Corporate Login Authentication
- Issue Reporting Form
- Dynamic Dashboard Metrics
- Category Progress Tracking
- Paginated Issue Table
- Local Storage Data Persistence
- Fully Responsive Layout
- Pure CSS (No Tailwind / No Frameworks)

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Custom Styling)
- Vanilla JavaScript
- Browser LocalStorage
